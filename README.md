Clean.CMD
One-click Windows cleaner. Deletes temp files, prefetch, and %temp% folders instantly.

No bloat. No spyware. Just runs the three commands Windows should run itself.

🚀 How to Use
Download clean.cmd

Right-click → Run as administrator

Done.

No install. No restart required (but recommended).

📁 What It Cleans
temp — temporary files

%temp% — user temp files

prefetch — old program traces

⚠️ Safe?
Yes. Only deletes what Windows marks as safe to remove. Same as doing it manually. Nothing personal, nothing critical.

📌 Why This Exists
Windows won't clean itself. This takes 2 seconds.
